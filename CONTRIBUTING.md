# Contributing to Grokden

Thanks for helping improve Grokden.

## Development setup

1. Install Node.js 20 or newer and Rust stable.
2. Install Tauri prerequisites for your platform.
3. Install the Grok CLI using the official xAI installer.
4. Run `npm install` in the repository root.
5. Run `npm run tauri dev` to start the desktop app in development mode.

## Pull requests

Keep changes focused. Run `npm run check` before submitting. Do not include secrets, tokens, API keys, passwords, personal file paths, machine-specific configuration, or private logs in any pull request.

## Reporting issues

Include your operating system version, Grokden version, whether Grok CLI is installed, and steps to reproduce the problem.
